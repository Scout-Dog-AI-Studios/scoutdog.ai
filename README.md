# Scout Dog AI Studios LLC

Company website for Scout Dog AI Studios LLC - a sole proprietorship building useful apps for web and mobile.

## Local Development

1. Install Ruby and Bundler
2. Install dependencies:
   ```bash
   bundle install
   ```
3. Run the development server:
   ```bash
   bundle exec jekyll serve
   ```
4. Open http://localhost:4000

## Deployment

This site is designed for GitHub Pages. Push to the `main` branch to deploy automatically.
